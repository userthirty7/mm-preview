Sharing

Share securely from one place

Your files live in one central hub. When you need to share, create a link with full control over who can access it and for how long.

Shareable Links

Generate a link for any file, track, or project. Share masters with the label, send lyrics to collaborators, or deliver finals to clients.

Password Protection

Add a password to any link. Only people with the password can access the content.

Access Limits

Set how many times a link can be opened. Once the limit is reached, the link expires automatically.
